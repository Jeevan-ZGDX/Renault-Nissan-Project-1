# CanteenX Meal Planner

CanteenX is a web application designed to help employees plan their weekly meals. Users can select meal types, choose specific days, and customize their meal selections. The application is built using React and TypeScript, providing a modern and interactive user experience.

## Features

- **Meal Selection**: Users can choose from a variety of meals for breakfast, lunch, dinner, and supper.
- **Weekly Planning**: The application allows users to plan meals for the entire week.
- **User-Friendly Interface**: The interface is designed to be intuitive and easy to navigate.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd canteenx
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the application:
   ```
   npm start
   ```

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **TypeScript**: A typed superset of JavaScript that compiles to plain JavaScript.
- **Lucide React**: A collection of icons for React applications.
- **Sonner**: A toast notification library for React.
